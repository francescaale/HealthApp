# HealthApp

A simple and safe app designed to support young adults with sexual and reproductive health. The app provides trusted information, private chat support, and local clinic access, all in one place.

# Tech Stack
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Google Services
- REST API
